# Rivne Dev

Rivne Dev is a small GitHub organization for developers from Rivne and the surrounding area.

The goal is simple: create a shared space for collaboration, experiments, and open-source projects maintained by local developers and friends.

## What we build

This organization hosts a mix of projects such as:

- experimental tools
- frontend and JavaScript utilities
- automation scripts
- pet projects
- collaborative learning projects

Most repositories here are informal and experimental. The main focus is learning, building, and sharing knowledge.

## Who is this for

- developers from Rivne
- friends working on side projects together
- contributors interested in small open-source experiments

## Contributing

If you are part of the organization, feel free to open issues, create pull requests, and experiment.

Small ideas and quick prototypes are welcome.

## Community

This organization exists primarily as a technical playground for collaboration between developers connected to Rivne.
